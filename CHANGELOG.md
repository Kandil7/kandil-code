## [0.1.0] - 2025-01-07

### Added
- Initial release of Kandil Code - Intelligent Development Platform
- Multi-language project generation (Flutter, Python, JavaScript, Rust)
- Intelligent CLI with context-aware commands
- Interactive TUI Studio with file navigation and AI chat
- Unified AI interface supporting local (Ollama) and cloud models (Claude, Qwen, OpenAI)
- Multi-Agent System with specialized agents for development tasks
- Project management with secure local storage
- Requirements elicitation and architectural design
- Code generation and review capabilities
- Professional role simulations (Project Manager, Business Analyst, Architect, Developer, QA)
- DevOps and Scrum simulation features
- Internationalization with RTL language support
- Accessibility features with WCAG compliance
- Cross-platform support (Linux, macOS, Windows)
- Git integration and version control
- Plugin system for extensibility
- Security features with OS keyring for API keys
- Configuration management
- Local model management for offline AI usage
- Cost tracking for AI usage
- Database integration with SQLite and Supabase sync
- Code analysis and refactoring tools
- Test generation and execution
